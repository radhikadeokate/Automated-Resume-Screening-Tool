# Automated Resume Screening Tool

An ML and NLP powered resume screening system that automatically ranks candidates against job descriptions with explainable results.

## Project Overview
This project focuses on designing an automated resume screening solution using Machine Learning and Natural Language Processing techniques. The system analyzes resumes and job descriptions, extracts skills and experience, and ranks candidates based on relevance.
